# Hackathon Video Submission Guide

## 60–90 second script

1. **Hook (5s):** "What if your dental clinic never missed a call?"
2. **Problem (10s):** Missed calls = lost patients. Front desk overwhelmed.
3. **Live demo (45s):**
   - Open `/demo/smilecare`
   - Speak in English: tooth pain → doctor recommendation → slot → price
   - Show Stripe payment link opening → pay with test card
   - Cut to dashboard: appointment **Confirmed**, deposit **Paid**
4. **CTA (10s):** "Reception.ai — ElevenLabs voice + Stripe payments. #ElevenHacks"

## Recording tips

- Split screen: demo page + dashboard
- Show Stripe success toast on landing hero for B-roll
- Use headphones to avoid echo
